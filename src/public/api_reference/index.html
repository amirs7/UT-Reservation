    Get List of Avaialable Rooms

     curl "http://213.233.176.40/available_rooms\
?start=2019-09-13T19:20:00&end=2019-09-13T19:40:00"

    const axios = require('axios');

axios.get('http://213.233.176.40/available_rooms' +
    '?start=2019-09-13T19:20:00&end=2019-09-13T19:40:00')
    .then(result => {
        console.log(result.data);
});

    Output:

    {
  "availableRooms":[
    801,

    810
  ]
}

    This endpoint can be used to retrieve the list of available rooms regarding the specified start and end date of the desired meeting.

    HTTP Request

    GET http://213.233.176.40/available_rooms

    Query Parameters

    Parameter Required Description
    start Yes Start time of the meeting for checking rooms availablity
    end Yes End time of the meeting for checking rooms availablity
    All date values should be provided with the 'YYYY-MM-DDTHH:mm:ss' format

    Errors

    Error Code Description
    400 Bad Request -- Invalid start or end date
    500 Internal Server Error -- Reservation System has been crashed.

    Reserve a room

    curl --header "Content-Type: application/json" \
  --request POST \
  --data '{"username":"rkhosravi","start": "2019-09-13T19:00:00","end": "2019-09-13T20:00:00"}' \
  "http://213.233.176.40/rooms/801/reserve"

    const axios = require('axios');

axios.post('http://213.233.176.40/rooms/801/reserve',
    {
        "username": "rkhosravi",
        "start": "2019-09-13T19:00:00",
        "end": "2019-09-13T20:00:00"
    }
).then(result => {
    console.log(result.data);
});

    Output:

    {
  "message": "Room 804 successfully reserved for:rkhosravi",
}

    This endpoint can be used to reserve a specific room.

    HTTP Request

    POST http://213.233.176.40/rooms/<room_id>/reserve

    URL Parameters

    Parameter Description
    room_id The id of the room to be reserved

    Body

    Property Required Description
    username Yes Username of the person who is reserving the room
    start Yes Start date of the reservation
    end Yes End date of the reservation

    Errors

    Error Code Description
    400 Bad Request -- Invalid start or end date, Required values not provided
    404 Not Found -- The specified room could not be found
    500 Internal Server Error -- Reservation System has been crashed
